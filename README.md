**Jumper Environment**
-

a simple custom environment for openai gym framework

**Installation**
1. clone the repository
2. `cd gym_jumper`
2. activate a python3 virtualenv [optional]
3. `pip install -e .`
4. `pip install "ray[rllib]"`

**Run a test**
- python gym_jumper/tests/ppo_stable_baseline.py
